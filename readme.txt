Разработанная программа позволяет экспортировать данные из Excel-файла с любой структурой данных, 
при этом достаточно просто указать в дескрипторе файла свойства данных, необходимых для извлечения, 
а также создать SQL-запрос, в котором могут быть произведены различные действия с извлеченными данными 
перед вставкой в базу.
